# EliteMindset ChatGPT App

Micro-action clarity coaching for ChatGPT users experiencing procrastination and overwhelm.

## Features
- State-based coaching (overwhelmed, stuck, ready_to_act, unclear_direction)
- Illustrated guidance with gold and black branding
- Personalized micro-action suggestions

## Deployment
Deployed on Render.com as MCP server for ChatGPT integration.
